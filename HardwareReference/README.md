# Hardware Reference Files

This directory contains reference implementations and header files for NeXT hardware components and system-level code.

## Files

### NextDimension Hardware
- **NextDimension1080p.h** - Header file for NextDimension 1080p graphics board
- **NextDimension1080p.m** - Objective-C implementation for NextDimension 1080p
- **NextDimension_Registers.h** - Register definitions for NextDimension hardware

NextDimension was NeXT's high-performance color graphics accelerator board, providing 32-bit true color display at resolutions up to 1920x1080.

### Mach Kernel
- **reconstructed_mach_kernel.h** - Reconstructed Mach kernel header file with system-level definitions

These files serve as historical reference material for understanding NeXT hardware architecture and low-level system programming on NeXTSTEP.
